FixIt

FixIt es una aplicación web en la que clientes y reparadores pueden ponerse en contacto de diferentes formas para solucionar y arreglar todo tipo de averías y fallos en una amplia gama de dispositivos tecnológicos, cómo móviles u ordenadores.

    Vistas

    Detalla aquí tus vistas. Usa enlaces a localhost:8080/ (SIN nombre de proyecto; correspondientes a arrancar la aplicación en modo spring-boot desde STS) para enlazar cada vista.
    • Home - Pantalla principal con una barra de búsqueda, en la que no se requiere login.
    • Contacto - Vista en la que se ofrecen vías de contacto al usuario.
    • Administrador - Vista para administradores.
    • Publicar anuncio - Vista que sirve para la descripción de un anuncio.
    • Registrarse - Página que permite crear nuevos usuarios en la base de datos.
    • Iniciar sesión - Vista que permite al usuario ingresar en la página con sus credenciales.
    • Salir - Vista para cerrar sesión.
    • Perfil - En esta vista se podrán ver todos los datos personales del usuario.
